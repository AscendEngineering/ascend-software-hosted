Installation Guide

  1. Double-click the AscendInstallerWindows.exe file to launch the installer
  2. Click Load Activation License and select the license file that was emailed to you
  3. Enter your drone's IP address (ensure both your PC and drone are on the same network)
  4. Provide the SSH password for installation access
  5. Click Retrieve Drone Info to verify the connection
  6. Click Install and wait for the process to complete
