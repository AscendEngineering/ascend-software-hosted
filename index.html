Link to our software installer

How to use:

  • The installer runs on Windows, so double click the .exe file
  • Click Load Activation License, and select the license file emailed to you
  • Enter the IP address of the drone (Ensure that your drone and the installer PC are on the same network, else it will not install)
  • Enter the SSH access password, this will be used to install the software only
  • Click Retrieve Drone Info
  • Then click Install, and it should just work
